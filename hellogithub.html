С помощью Git программисты и разработчики ориентируются в коде и отслеживают изменения. Git поиогает вернуть файлы в исходное состояние и видеть, внесенные изменения в определенный период. Разработчик выполняет разные команды (например, commit, push), а все измения синхранизируются с центральным репозиторием. Git- это система контроля версий, а GitHub- онлайн-сервис, по сути социальная сеть. Одна из основных целей GitHub- быть единым местом для проектов с исходным кодом. Предполагается, что пользователь делится чем-то полезным, а другие люди смогут учавствовать в разработке.
